# Atividade Flexbox e Grid

## Objetivo
Praticar a criação de layouts utilizando CSS Flexbox e CSS Grid.

## Exercícios

### Exercício 1
Navbar criada utilizando Flexbox com alinhamento horizontal.

### Exercício 2
Cards organizados com Flexbox utilizando flex-wrap.

### Exercício 3
Galeria criada utilizando CSS Grid com colunas e linhas.

### Exercício 4
Layout utilizando grid-template-areas para organizar os elementos.

### Exercício 5
Estrutura principal criada com Grid e alinhamento interno com Flexbox.
